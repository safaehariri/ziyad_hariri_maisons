# Plan du site - Ziyad Hariri Maisons

## Objectif

Créer un site vitrine moderne, clair et rassurant pour une entreprise de construction clé en main. Le message principal est simple : le client apporte son plan, l'entreprise pilote le chantier jusqu'à la remise des clés.

## Structure

1. Accueil
   - Hero avec slogan, explication courte et deux appels à l'action.
   - Mise en avant de la promesse : budget, délais, qualité, suivi.

2. À propos
   - Présentation de l'entreprise comme partenaire fiable.
   - Accent sur la transparence, la coordination et la qualité d'exécution.

3. Services
   - Cartes réutilisables pour villas, maisons R+2/R+3, gros oeuvre, finitions, suivi de chantier et clé en main.

4. Comment ça marche
   - Parcours en 5 étapes, du plan client à la remise des clés.

5. Réalisations
   - Galerie de projets avec images/cartes placeholder.
   - Les commentaires HTML indiquent où remplacer par les vraies photos de chantier.

6. Contact
   - Formulaire nom, téléphone, type de projet, message.
   - CTA WhatsApp, téléphone et localisation placeholders.

## Choix techniques

- Site statique en HTML, CSS et JavaScript léger.
- HTML sémantique.
- CSS organisé avec variables, composants réutilisables et media queries.
- Responsive mobile/desktop.
- Image hero générée et enregistrée dans `assets/hero-construction.png`.
